# AI in SDLC (Software Development Lifecycle)

AI verandert hoe software wordt ontworpen, ontwikkeld, getest en beheerd. Van requirements tot release: elk stadium van de SDLC is beïnvloedbaar met AI.

---

## 1. Overzicht: AI in elke SDLC-fase

| Fase               | AI-toepassing                                     |
|--------------------|---------------------------------------------------|
| Requirements       | Genereren van user stories en acceptatiecriteria  |
| Design             | Architectuursuggesties, diagramgeneratie          |
| Development        | Code completion, testsuggesties, refactoringen    |
| Testing            | Testcase generatie, fuzz testing, log analyse     |
| Deployment         | Automatiseren van CI/CD-pipelines                 |
| Monitoring         | Incidentdetectie, changelog generatie, logging    |
| Maintenance        | Root cause analyse, code-analyse, feature tracing |

---

| Development       | GitHub Copilot, Cody, Replit, Tabnine             |
| Testing           | Testim, Codegen met LLM, Cypress met GPT-integratie|
| Deployment        | GitHub Actions, Terraform met prompts             |
| Monitoring        | Datadog + GPT, Traceloop, Honeycomb AI            |

---

## 3. Prompting voor SDLC

Gebruik specifieke promptstructuren per fase:

- **Requirements:** “Schrijf user stories voor feature X met [doelgroep], [acceptatiecriteria] en [edge cases].”
- **Code:** “Schrijf in Python een functie die X doet, met logging en tests.”
- **Tests:** “Genereer 5 edge case testscenario’s voor input Y.”
- **Refactor:** “Optimaliseer onderstaande code voor performance en leesbaarheid.”
- **Debug:** “Wat zijn mogelijke oorzaken van deze foutmelding [log output]?”

---

## 6. Valkuilen

- Te snel in productie zonder evaluatie
- Te weinig SDLC-kennis bij AI-ontwikkelaars
- Geen feedbackloop: AI leert niet bij
- AI als gimmick → weinig adoption
- Alleen GitHub Copilot → te eenzijdig

---

## 7. Conclusie

AI is geen extraatje in SDLC, maar een krachtig vliegwiel.  
Wie SDLC + AI combineert, ontwikkelt sneller, met hogere kwaliteit én meer plezier.
